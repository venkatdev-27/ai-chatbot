# AI Realtime Chatbot

A full-stack MERN application (MongoDB, Express, React, Node.js) featuring real-time communication via Socket.io and AI capabilities using Google Gemini.

## Features
- Real-time messaging
- AI Chatbot integration (Gemini)
- User Authentication (JWT)
- Media Sharing (Images, Videos)

## Structure
- `/frontend`: React application (Vite)
- `/backend`: Node.js/Express API & Socket server
